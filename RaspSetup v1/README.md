RaspSetup v1
Initial stable release – basic system setup
Main Features
Whiptail-based interactive menu
System information detection:
OS version
Raspberry Pi model
Safe dpkg repair function before installations
System update & upgrade
Optional application installation:
Sysbench
Stress-ng
Neofetch
Pi-Apps
RealVNC
Optional access to raspi-config
Optional reboot after completion
Key Characteristics
Simple and lightweight
Focused on basic benchmarking & utilities
Manual progress bars
Strong emphasis on preventing broken dpkg states
Limitations
Limited application list
Minimal automation
No security or server management tools
