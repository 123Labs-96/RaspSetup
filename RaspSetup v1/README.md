### 🔹 RaspSetup v1 — CHANGELOG

#### Features
- System information detection (OS & board model)
- Automatic `dpkg` repair before installations
- System update & upgrade
- Optional installation of:
  - Sysbench
  - Stress-ng
  - Neofetch
  - Pi-Apps
  - RealVNC
- Optional access to `raspi-config`
- Optional reboot after completion

#### Intended For
- Beginners
- Fresh Raspberry Pi installations
- Basic performance testing
