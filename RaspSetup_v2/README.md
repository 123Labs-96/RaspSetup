### 🔹 RaspSetup v2 — CHANGELOG

#### New & Improved
- Fully modular function-based architecture
- Real-time progress gauges for `apt`
- Advanced application detection
- Expanded application list:
  - Sysbench
  - Stress-ng
  - Neofetch
  - Pi-Apps
  - RealVNC
  - Htop
  - UFW Firewall
  - Node.js & NPM
  - Webmin
  - Fail2Ban
- Official installation scripts for:
  - Pi-Apps (run as regular user)
  - Webmin (Bookworm & Trixie compatible)
- Optional automatic Fail2Ban configuration for Webmin
- Improved error handling and logging

#### Intended For
- Raspberry Pi servers
- Homelab environments
- Advanced users and administrators
